Задание 1

В ранее выполненном домашнем задании о справочнике:

	Создать класс СПРАВОЧНИК со следующими полями:
		1. Название фирмы;
		2. Владелец;
		3. Телефон;
		4. Адрес;
		5. Род деятельности.
	Реализовать следующие возможности:
		1. Поиск по названию;
		2. Поиск по владельцу;
		3. Поиск по номеру телефона;
		4. Поиск по роду деятельности;
		5. Показ всех записей и добавление.
	Вся информация, должна сохранятся в файле, должна
	быть реализована возможность добавления новых данны

Реализовать функции сохранения/считывания данных из файла через потоки.

Дополнительное задание.

Реализовать функцию рекурсивного поиска файлов и папок относительно начальной 
директории, которая указывается пользователем при запуске программы
